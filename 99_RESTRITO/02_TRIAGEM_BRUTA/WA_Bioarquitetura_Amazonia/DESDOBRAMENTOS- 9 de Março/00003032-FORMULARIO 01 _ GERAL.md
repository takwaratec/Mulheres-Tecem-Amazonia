| **Apresentação**                           | **APRESENTAÇÃO GERAL**                          |
|                                            |                                                 |
|                                            | Amazônia é um dos principais sistemas           |
|                                            | reguladores do clima global e abriga uma        |
|                                            | sociobiodiversidade singular, sustentada por    |
|                                            | modos de vida, práticas produtivas e            |
|                                            | conhecimentos tradicionais que historicamente   |
|                                            | garantem a manutenção da floresta em pé. Diante |
|                                            | do avanço de modelos econômicos predatórios e   |
|                                            | da intensificação das crises climáticas e       |
|                                            | sociais, torna-se estratégico afirmar, para a   |
|                                            | região, um horizonte de desenvolvimento baseado |
|                                            | na **(nova) bioeconomia**: uma economia que     |
|                                            | transforma a biodiversidade e os saberes        |
|                                            | associados em **valor local**, combinando       |
|                                            | conservação, inovação, justiça social e         |
|                                            | regeneração territorial.                        |
|                                            |                                                 |
|                                            | O projeto de pesquisa **"Mãos que Tecem a       |
|                                            | Floresta: O Elo Feminino entre a                |
|                                            | Biodiversidade, a Bioarquitetura e a Nova       |
|                                            | Bioeconomia"** parte do reconhecimento de que   |
|                                            | as cadeias não madeireiras --- **castanhas,     |
|                                            | açaí e artesanato** --- constituem pilares da   |
|                                            | economia das florestas e são fortemente         |
|                                            | sustentadas pelo trabalho e pela inteligência   |
|                                            | coletiva das **mulheres amazônicas**. Ao mesmo  |
|                                            | tempo, essas cadeias geram **insumos e resíduos |
|                                            | agroextrativistas** ainda subaproveitados       |
|                                            | (cascas, fibras, sementes, caroços, endocarpos, |
|                                            | biomassas e subprodutos), que podem integrar    |
|                                            | rotas tecnológicas de baixo impacto e alto      |
|                                            | valor agregado.                                 |
|                                            |                                                 |
|                                            | Nesse contexto, a pesquisa propõe articular     |
|                                            | biodiversidade e inovação por meio da           |
|                                            | **bioarquitetura** --- entendida como o campo   |
|                                            | que aplica princípios bioclimáticos, materiais  |
|                                            | de base biológica e soluções construtivas       |
|                                            | adequadas ao território --- incorporando        |
|                                            | **resíduos e coprodutos das cadeias de          |
|                                            | castanhas e açaí**, bem como o **bambu**, como  |
|                                            | matéria-prima renovável e versátil. A           |
|                                            | bioarquitetura, aqui, não é apenas uma técnica  |
|                                            | construtiva: é uma estratégia de **habitação    |
|                                            | sustentável**, soberania material, saúde        |
|                                            | ambiental e melhoria concreta das condições de  |
|                                            | vida, conectando produção, moradia e            |
|                                            | conservação.                                    |
|                                            |                                                 |
|                                            | **3. JUSTIFICATIVA GERAL**                      |
|                                            |                                                 |
|                                            | A justificativa central do projeto reside na    |
|                                            | urgência de **fortalecer e sofisticar as        |
|                                            | cadeias produtivas da sociobiodiversidade** sem |
|                                            | reproduzir a lógica da "bioeconomia de          |
|                                            | commodities" (concentradora, padronizadora e    |
|                                            | dependente de grande escala). A **nova          |
|                                            | bioeconomia** demandada para a Amazônia deve    |
|                                            | ser **diversa, territorializada e inclusiva**:  |
|                                            | capaz de agregar valor no local, reduzir        |
|                                            | vulnerabilidades logísticas e comerciais, e     |
|                                            | criar alternativas econômicas compatíveis com a |
|                                            | integridade dos ecossistemas e com os direitos  |
|                                            | das populações tradicionais.                    |
|                                            |                                                 |
|                                            | Entretanto, embora as mulheres sejam            |
|                                            | protagonistas no manejo, na seleção, no         |
|                                            | processamento, no artesanato e na transmissão   |
|                                            | de saberes ligados à floresta, persistem        |
|                                            | **barreiras estruturais**: acesso limitado a    |
|                                            | tecnologias apropriáveis, infraestrutura,       |
|                                            | crédito, assistência técnica, conectividade,    |
|                                            | regularização produtiva e canais de             |
|                                            | comercialização justos. Essa assimetria diminui |
|                                            | a renda, aumenta a dependência de               |
|                                            | atravessadores e restringe o potencial de       |
|                                            | inovação comunitária. Fortalecer o protagonismo |
|                                            | feminino nessas cadeias é, portanto, uma ação   |
|                                            | direta de **equidade de gênero**, redução de    |
|                                            | desigualdades e **sustentabilidade              |
|                                            | territorial**, pois amplia a capacidade de      |
|                                            | governança local sobre recursos, processos      |
|                                            | produtivos e destinos do excedente econômico.   |
|                                            |                                                 |
|                                            | Para enfrentar essas lacunas, o projeto se      |
|                                            | ancora na convergência entre **Bioeconomia**,   |
|                                            | **Economia Solidária (EcoSol)** e **Tecnologia  |
|                                            | Social (TS)** --- tríade que organiza a         |
|                                            | proposta como um arranjo coerente entre *o que* |
|                                            | produzir, *como* transformar e *para quem*      |
|                                            | gerar benefícios:                               |
|                                            |                                                 |
|                                            | - **Bioeconomia (base material e ecológica):**  |
|                                            |   orienta o uso sustentável de recursos         |
|                                            |   biológicos renováveis, preservando ciclos     |
|                                            |   naturais e mantendo a floresta em pé. No      |
|                                            |   projeto, ela inclui tanto os produtos         |
|                                            |   principais (castanhas, açaí, artesanatos)     |
|                                            |   quanto a valorização de **coprodutos e        |
|                                            |   resíduos** como parte de uma bioeconomia      |
|                                            |   **circular** e de baixo desperdício.          |
|                                            |                                                 |
|                                            | - **Tecnologia Social (método de inovação       |
|                                            |   apropriável):** viabiliza melhorias concretas |
|                                            |   no beneficiamento, na segurança, na qualidade |
|                                            |   e na gestão, por meio de soluções **de baixo  |
|                                            |   custo, co-desenvolvidas e replicáveis**,      |
|                                            |   adequadas à realidade comunitária (por        |
|                                            |   exemplo: secagem eficiente, prensas,          |
|                                            |   adaptações ergonômicas, protocolos simples de |
|                                            |   controle de qualidade, organização de         |
|                                            |   produção e registro). A TS também sustenta a  |
|                                            |   experimentação de usos construtivos para      |
|                                            |   biomateriais derivados das cadeias de         |
|                                            |   castanhas e açaí e para o **bambu**,          |
|                                            |   promovendo autonomia técnica e aprendizagem   |
|                                            |   coletiva.                                     |
|                                            |                                                 |
|                                            | - **Economia Solidária (governança e            |
|                                            |   justiça):** define a forma de organização do  |
|                                            |   trabalho e da riqueza com **autogestão,       |
|                                            |   cooperação e repartição equitativa**,         |
|                                            |   protegendo o valor social do trabalho         |
|                                            |   feminino e fortalecendo redes comunitárias.   |
|                                            |   Ao orientar a comercialização e a tomada de   |
|                                            |   decisão, a EcoSol reduz assimetrias e amplia  |
|                                            |   a capacidade de reinvestimento local.         |
|                                            |                                                 |
|                                            | A integração desses pilares permite que a       |
|                                            | inovação não seja extrativa nem excludente, mas |
|                                            | um processo de **tecnologia com vínculo         |
|                                            | social** --- uma *tecnologia social* baseada em |
|                                            | confiança, cooperação e coautoria --- na qual   |
|                                            | as mulheres não são apenas beneficiárias, e sim |
|                                            | **pesquisadoras do próprio território**,        |
|                                            | liderando soluções para produção, renda,        |
|                                            | habitação e conservação. Essa "harmonia entre   |
|                                            | saber tradicional e inovação técnica" expressa  |
|                                            | uma forma de resiliência territorial feminina:  |
|                                            | ao mesmo tempo em que protege a biodiversidade, |
|                                            | reorganiza economias locais e sustenta uma      |
|                                            | ética do cuidado (com a floresta, com a         |
|                                            | alimentação, com a saúde e com a comunidade).   |
|                                            |                                                 |
|                                            | Além disso, ao aproximar cadeias                |
|                                            | agroextrativistas da **bioarquitetura**, o      |
|                                            | projeto amplia o sentido de valor agregado: não |
|                                            | apenas vender melhor, mas **viver melhor** no   |
|                                            | território. Resíduos antes descartados podem    |
|                                            | integrar novas aplicações (materiais,           |
|                                            | componentes, acabamentos, soluções              |
|                                            | bioclimáticas), enquanto o bambu oferece        |
|                                            | alternativas renováveis para estruturas leves,  |
|                                            | painéis, sombreamento e sistemas construtivos   |
|                                            | compatíveis com a realidade amazônica. Essa     |
|                                            | ponte entre produção e moradia fortalece a      |
|                                            | sustentabilidade de forma sistêmica: renda,     |
|                                            | trabalho digno, autonomia técnica, redução de   |
|                                            | impactos e melhoria do habitat.                 |
|                                            |                                                 |
|                                            | Em síntese, o projeto se justifica por propor   |
|                                            | uma rota aplicada e territorializada para       |
|                                            | **fortalecer o protagonismo das mulheres e a    |
|                                            | sustentabilidade das cadeias de castanha, açaí  |
|                                            | e artesanato**, integrando **saberes            |
|                                            | tradicionais, pesquisa aplicada e tecnologias   |
|                                            | sociais**, impulsionando a **bioeconomia e a    |
|                                            | nova bioeconomia** amazônicas, promovendo a     |
|                                            | **economia solidária** e contribuindo para a    |
|                                            | **conservação da floresta em pé** e para a      |
|                                            | **habitação sustentável**.                      |

1.  **[ALINHAMENTO COM POLÍTICAS PÚBLICAS]{.smallcaps}**

> O projeto está alinhado com ações presentes nas principais políticas
> públicas socioambientais, programas e planos de ação de governo como:

a)  **Política de Concessões Florestais** (Lei 11.284/2006), **Programa
    Nacional de Florestas** (PNF) e o **Código Florestal Brasileiro**
    (Lei 12.651/2012) na medida em que produzirá informações científicas
    e tecnológicas sobre espécies de madeira pouco conhecidas pelo
    mercado tornando mais atraente a manutenção da floresta do que o uso
    alternativo do solo, gerando novas alternativas de renda para a
    população local e diminuindo a pressão sobre espécies em extinção,
    bem como, agregando valor ao produto floresta e estimulando o
    desenvolvimento da atividade florestal sustentável. Há ainda a
    valorização da madeira proveniente das áreas de concessões
    florestais, pois, com a caracterização de novas espécies haverá
    agregação de valor ao produto madeireiro -- certificado e com cadeia
    de custódia (rastreabilidade da madeira) tornando ainda mais
    vantajosas as concessões florestais no país.

> O projeto tem por escopo aumentar a rentabilidade dos produtos
> advindos de manejo florestal -- concessões florestais e planos de
> manejo, fomentando as atividades produtivas sustentáveis. Demonstra
> alinhamento também com as políticas públicas previstas no Código
> Florestal Brasileiro, pois com as pesquisas geradas haverá estímulo ao
> desenvolvimento da atividade florestal, tornará a floresta em pé
> economicamente mais atraente do que o uso alternativo do solo e
> incentivará a geração de alternativas de renda com os estudos das
> novas espécies, atendendo o objetivo previsto no art. 2º do novo
> Código Florestal*: "fomentar a pesquisa científica e tecnológica na
> busca da inovação para o uso sustentável e preservação das florestas e
> demais formas de vegetação nativa*".

b)  **Programa Nacional de Habitação Rural** (PNHR), os sistemas
    construtivos que serão desenvolvidos têm como público alvo o mesmo
    público do PNHR que procura atender: agricultores familiares,
    trabalhadores rurais, quilombolas, extrativistas, ribeirinhos e
    populações tradicionais com foco nas populações tradicionais de
    baixa renda localizadas na região Norte, em especial, as que
    circundam as concessões de florestas públicas. O desenvolvimento de
    novos sistemas construtivos e sustentáveis tem como foco adaptações
    às características locais e a confecção de normas técnicas, a fim de
    padronizar o uso da madeira na construção de obras de interesse
    social (habitações populares, postos de saúde, escolas, etc.) e
    consequentemente, na construção civil em geral. Outro beneficiário
    direto será o próprio governo, pois os métodos construtivos também
    poderão ser utilizados para edificação de prédios públicos nas
    localidades.

c)  **Política Nacional de Mudança do Clima** (Lei nº12.187/2007), o
    resultado das pesquisas irá também fomentar a redução das emissões
    de gases de efeito estufa, tendo em vista que ao se empregar
    técnicas construtivas em alvenaria gera-se consumo de produtos não
    renováveis como aço e cimento e ao se empregar a madeira, além de se
    fomentar o uso de produto renovável, será utilizado material com
    menor potencial de emissão com o uso de novas espécies de madeira e
    novos produtos derivados. Essa externalidade positiva está alinhada
    à Lei nº12.187/2007, cujo art. 5º traz as diretrizes dessa política
    que, entre outras, visa a promoção e o desenvolvimento de pesquisas
    científico-tecnológicas, e a difusão de tecnologias, processos e
    práticas.

d)  **Plano de Prevenção e Controle ao Desmatamento na Amazônia Legal**
    (PPCDAM) e o **Plano Amazônia Sustentável** (PAS). O alinhamento do
    projeto com o PPCDAM insere-se no eixo "Fomento às Atividades
    Produtivas Sustentáveis", mais especificamente nos incisos IV e V,
    quando é informado que se tem por interesse apoiar as atividades
    econômicas desenvolvidas a partir do uso sustentável da floresta,
    bem como da conservação e uso sustentável da biodiversidade, criando
    um mercado regional e nacional para consumo da madeira das
    concessões. Com o aumento da rentabilidade do manejo florestal e o
    fomento às atividades produtivas sustentáveis, o projeto promoverá a
    viabilidade da cadeia produtiva da madeira, constituindo alternativa
    para valorizar a floresta em pé e diminuir o desmatamento, e gerará
    e difundirá conhecimento tecnológico, científico e de inovação sobre
    este precioso recurso da Amazônia. O projeto ainda se alinha ao
    Plano ao gerar incremento às concessões, fortalecendo a economia
    florestal, particularmente das cadeias produtivas derivadas de
    Manejo Florestal Sustentável, a difusão do conhecimento científico e
    tecnológico sobre as espécies madeireiras e seus diversos usos
    gerará inovação e agregará valor ao produto florestal, promovendo o
    desenvolvimento regional de onde as espécies serão extraídas e
    valorização da cadeia produtiva, o que é um dos resultados esperados
    pelo PPCDAM. No mesmo sentido, o conhecimento de outras espécies e
    seus usos será capaz de diminuir a pressão pela procura daquelas
    muito ameaçadas de extinção (conforme lista da CITES -- Lista de
    Espécies Ameaçadas de Extinção ou com Restrições).

> Quanto ao alinhamento do projeto com o Plano Amazônia Sustentável --
> PAS, o projeto atende aos incisos I, II, X e XI. O PAS atua em nível
> estratégico e aponta 4 eixos dos quais o 2º é o que mais se enquadra
> ao projeto, ou seja, "Desenvolver a Produção Sustentável com Inovação
> e Competitividade". Esse Plano tem ainda como eixo central o
> "Desenvolvimento Regional" e o projeto em questão busca, pela pesquisa
> de novas espécies madeireiras e dos novos métodos construtivos
> desenvolvidos, atender a população local para que ela possa se
> beneficiar dos programas de governo para financiamento de moradias em
> madeira. Ainda, o projeto atenderá os objetivos específicos do PAS
> alíneas: b) fomentar atividades econômicas pautadas no uso sustentável
> dos recursos naturais com inovação tecnológica; d) fortalecer a
> inclusão social, uma vez que as informações serão utilizadas para
> fomentar projetos habitacionais locais; e) embasar a construção de um
> novo modelo de financiamento na Amazônia voltado para a redução das
> desigualdades sociais e regionais e para o uso sustentável dos
> recursos naturais. O projeto também se alinha com as diretrizes
> estratégicas do PAS ao incentivar a (ii) Produção Sustentável com
> Inovação e Competitividade.

e)  **Política Nacional de Resíduos Sólidos** (Lei nº 12.305/2010), uma
    das linhas de pesquisa que será desenvolvida no âmbito desse projeto
    busca promover o aproveitamento de resíduos florestais para produção
    de compostos madeira/plástico, utilizando resíduos de serrarias e
    plásticos reciclados oriundo da coleta seletiva decorrente dos
    Planos Municipais de Resíduos Sólidos. Serão beneficiários diretos
    dessa atividade as associações e as cooperativas de reciclagem, com
    a possibilidade de construção de cadeias produtivas geradoras de
    trabalho e renda em nível local e regional. Com o desenvolvimento
    das pesquisas será possível identificar as espécies, realizar
    ensaios estruturais, analisar a durabilidade natural da madeira e
    seu comportamento na secagem, estudar o potencial energético dos
    resíduos, identificar os melhores usos dos produtos florestais e
    identificar quais outros compostos de resíduos sólidos poderão ser
    agregados à madeira e assim serem desenvolvidos novos produtos.

2.  **[ANEXO II CRITÉRIOS TRANSVERSAIS OBSERVADOS PELO FUNDO
    AMAZÔNIA]{.smallcaps}**

*Critérios transversais são temas "horizontais" que devem ser abordados
de uma maneira integrada ao invés de serem tratados como assuntos
isolados. No contexto do Fundo Amazônia são considerados como assuntos
transversais "redução da pobreza" e "equidade de gênero".*

8.  **Redução da pobreza: avalia em que medida o projeto contribuirá
    para alternativas dignas de trabalho e renda na Amazônia Legal, que
    tenham como preocupação central a sustentabilidade ambiental e
    social**

pobreza, a inclusão social e a melhoria nas condições de vida de seus
beneficiários (principalmente comunidades tradicionais, assentados e
agricultores familiares) *(até 1000 caracteres, incluindo espaços)*.

9.  **Equidade de gênero: a equidade de gênero como critério transversal
    contempla em que medida o projeto considerou os interesses
    diferentes de homens e mulheres e integrou aspectos de equidade de
    gênero nas suas intervenções**

Descreva, se aplicável, como o projeto integrou questões de gênero nas

equidade de gênero *(até 1000 caracteres, incluindo espaços)*.
