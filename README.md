# Mulheres Que Tecem a Floresta 
**Consórcio Interinstitucional UnB / UFAC / UFRR**

## <img src='docs/assets/patterns/square_06_black.svg' width='22px'> Sobre o Projeto
Iniciativa voltada ao empoderamento econômico de mulheres amazônidas através da bioeconomia descentralizada, silvicultura de bambu nativo (*Guadua* spp.), e saneamento ecológico soberano. O projeto é estruturado em polos comunitários e canteiros-escola, integrando tecnologias de baixo impacto e alta eficiência térmica.

## <img src='docs/assets/patterns/square_12_black.svg' width='22px'> Tecnologias e Invenções (T01-T12)
O ecossistema técnico do projeto é composto por 12 invenções complementares, unificadas pelo **[Coração Térmico Mulheres que Tecem a Floresta](04_GESTAO_OPERACIONAL/01_FLUXO_TECNICO/TAK-MEM-MASTER_CORACAO-TERMICO.md)** (0-bar):
- **T01/T02**: Biorrefinaria e Reator Central.
- **T09**: Escariador Interno de Bambu.
- **T10**: Balsa-Catamarã a Vapor de Baixa Pressão.
- **T11**: Domo-Fábrica de Bio-Compósitos.
- **T12**: Banheiro Seco Modular (BSM).

## <img src='docs/assets/patterns/square_04_green.svg' width='22px'> Governança e Soberania de Dados (IA-Free)

O projeto opera sob o protocolo de **Soberania de Dados Territorial**, garantindo que a inteligência territorial, os censogramas das **1.150 famílias** e o conhecimento tradicional (TK) permaneçam sob controle exclusivo do Consórcio e das comunidades.

- **Licenciamento Híbrido**: Documentação técnica e científica são publicadas sob **CC BY 4.0**. Metadados sensíveis e registros de campo são protegidos por **Sigilo e Confidencialidade (NT-GOV-001)**.
- **Proteção Construtiva**: É proibida a extração automatizada de dados deste repositório para treinamento de modelos de IA comerciais externos, em conformidade com as diretrizes de soberania do território.
