# Regra de Versionamento Institucional (v1.0)

Este protocolo define como as versões dos arquivos devem ser gerenciadas em toda a coleção do Projeto Mulheres Que Tecem a Floresta.

## Padrão de Versionamento

1. **Versão Institucional**: Toda a coleção atual deve refletir a **Versão 1.0 (WTF)**.
2. **Data de Lançamento**: A data de transição para o novo padrão institucional é **18 de março de 2026**.
3. **Atualização em Lote**: Mudanças estruturais em memoriais ou protocolos exigem a atualização de versão em lote para manter a paridade documental.
4. **Escrita no Frontmatter**:
    - O campo `version` deve ser `"1.0"`.
    - O campo `date` deve refletir a data da consolidação.
    - Metadados recomendados: `Versão: 1.0 | Data: 2026-03-18`.

## Procedimento de Atualização

- **Trigger**: Mudanças em especificações ou governança do Consórcio UnB/UFAC/UFRR.
- **Ação**: Garantir que novos documentos sigam o prefixo `WTF_` (Governança) ou `ENG_` (Técnico).
