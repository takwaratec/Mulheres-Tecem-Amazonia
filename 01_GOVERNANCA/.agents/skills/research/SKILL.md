# Research Skill

Esta Skill deve ser aplicada a relatórios de pesquisa, análises laboratoriais, estudos de campo e dados de monitoramento (BSM/BER).

## Componentes Estratégicos
- **BSM (Bambu-Saneamento-Manejo)**: O tripé de sustentação da tecnologia social.
- **BSM (Banheiro Seco Modular)**: O produto/tecnologia para áreas terrestres.
- **BER (Banheiro Ecológico Ribeirinho)**: Adaptação da tecnologia para comunidades em áreas de várzea e rios.
- **Fitorremediação**: Dados de eficiência na remoção de contaminantes via sistemas vegetais.
- **Materiais**: Testes de resistência mecânica de compósitos de bambu + PU Vegetal.

## Integridade de Dados
- **Ciência Aberta**: Todos os dados brutos devem estar referenciados ou disponíveis para auditoria.
- **Metodologia**: Clara, replicável e contextualizada na biologia amazônica.
- **Referências**: Uso rigoroso de citações acadêmicas e links internos para a coleção Institucional.

## Verificação
- Revisão por pares internos (equipe técnica Institucional).
- Cruzamento de dados com a matriz de correlação da plataforma.
