# 📒 Catálogo de Metadados e Índices: Projeto WTF

Este diretório centraliza a inteligência e a organização bibliográfica do projeto **Mulheres Que Tecem a Floresta**. Ele serve como a ponte entre o acervo bruto e as análises estratégicas para o dossiê BNDES.

## Conteúdo do Diretório

### 1. [Índice Mestre (WTF_INDEX_ACERVO.md)](./WTF_INDEX_ACERVO.md)
O ponto central de navegação. Vincula cada título da bibliografia aos seus:
- **Transcritos Brutos**: Conteúdo extraído originalmente dos PDFs/Docs.
- **Resumos Estratégicos**: Análises de P&D com foco em Defesa do Projeto e Norteamento de CAPEX.

### 2. [Bibliografia Consolidada (WTF_BIBLIOGRAPHY_CONSOLIDATED.md)](./WTF_BIBLIOGRAPHY_CONSOLIDATED.md)
Lista completa e higienizada das obras de referência do projeto, catalogadas por título, autor e relevância temática.

## Protocolo de Atualização
- **Frequência**: O índice mestre deve ser atualizado sempre que novos transcritos ou resenhas forem gerados.
- **Ferramenta**: Utilize o script `.agents/scripts/generate_index.py` para sincronizar os links automaticamente.
- **Critério**: Apenas materiais técnicos e científicos validados devem constar neste catálogo. Documentos administrativos ou de triagem bruta (quarentena) não são indexados aqui.
