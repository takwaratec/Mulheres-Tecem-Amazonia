# Inventário Cirúrgico: Orçamento Mulheres que Tecem a Floresta 1.0 (Ref. Mestre Takwara)

Este documento consolida a alocação de recursos do **Projeto Mulheres Que Tecem a Floresta **, baseada na arquitetura técnica de R$ 25,3M consolidada para 48 meses.

## 1. Resumo Macroeconômico

| Categoria | Valor (R$) | % | Observação |
| :--- | :--- | :--- | :--- |
| **CAPEX Industrial/Obras** | 6.964.240 | 27% | Máquinas, Obras, Frota, Fretes, Licenciamento |
| **OPEX Operacional** | 18.416.717 | 73% | RH (CLT + encargos), Bolsas, Insumos, MRV, Reservas |
| **TOTAL GERAL** | **25.380.957** | **100%** | Auditoria Mulheres que Tecem a Floresta 1.0 |
