# Guia de Orçamento e Desenvolvimento: Moldes de Precisão em PP

Este guia orienta o diálogo técnico com fornecedores de usinagem CNC e ferramentarias para o desenvolvimento dos moldes do Sistema Institucional.

## 1. Especificações para Orçamento (Prompt CNC)

Utilize o texto abaixo para solicitar cotações a empresas de usinagem de plásticos de engenharia:

> **Solicitação de Orçamento Técnico — Matriz Colmeia Modular Eco-PU**
> 
> Procuro orçamento para kit de moldes em **Polipropileno (PP) Natural de 20mm**. Requisito de acabamento interno com rugosidade **Ra < 0,8 µm** (polimento espelhado para desmolde técnico).
> 
> **Configuração Sugerida:**
> *   01 Base: PP 20mm (1050 x 550 mm) com 20 furos escareados para M10.
> *   20 Machos Cônicos: Usinados em torno Ø62mm(base) a Ø60mm(topo), H=55mm.
> *   Grelha de Paredes: Sistema "pente-fino" em PP 15mm, altura 50mm.
> *   Tampa: PP 20mm com furos de alívio e guia para os pinos.
> 
> **Local de Entrega**: Rio Branco, AC (Considerar frete e embalagem para longa distância).

## 2. Tabela Comparativa: Alumínio vs. Polipropileno (PP)

O Polipropileno foi selecionado como material padrão por sua performance superior com a química de mamona:

| Característica | Alumínio Fundido | Polipropileno (PP) |
| :--- | :--- | :--- |
| **Antiaderência** | Baixa (exige muito desmoldante) | **Altíssima (natural)** |
| **Acabamento** | Rugoso/Fosco | **Espelhado/Brilhante** |
| **Peso** | Pesado | **Leve** |
| **Custo Inicial** | Alto (ferramental/fundição) | **Médio (chapas)** |
| **Manutenção** | Oxida com a umidade | **Inerte/Lavável** |

## 3. Passo a Passo para o Desenvolvedor

1.  **Seleção do Parceiro**: Priorizar empresas com experiência em usinagem de plásticos de engenharia (PP, PEAD, UHMW).
2.  **Validação de Protótipo**: Iniciar pelo molde de tijolo comum (190x90x50mm) em colmeia para validar a cura e o tempo de desmolde.
3.  **Protocolo de Limpeza**: Treinar a equipe no uso exclusivo de **Álcool 96 ou Isopropílico**. Proibir Thinner, Acetona ou Tolueno que degradam o polimento do PP.
4.  **Escalabilidade**: O sistema modular permite que as cavidades sejam multiplicadas conforme a demanda da obra, mantendo o mesmo padrão de pinos ejetores.
