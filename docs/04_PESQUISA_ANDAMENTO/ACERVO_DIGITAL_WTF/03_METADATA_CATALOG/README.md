# <img src="../../../../../assets/icons/human_01_red.svg" width="18px"> Catálogo de Metadados e Índices: Projeto Mulheres que Tecem a Floresta

Este diretório centraliza a inteligência e a organização bibliográfica do projeto **Mulheres Que Tecem a Floresta**. Ele serve como a ponte entre o acervo bruto e as análises estratégicas para o dossiê BNDES.

## Conteúdo do Diretório

### 1. [Índice Mestre (WTF_INDEX_ACERVO.md)](WTF_INDEX_ACERVO.md)
O ponto central de navegação. Vincula cada título da bibliografia aos seus:
- <img src="../../../../../assets/icons/human_13_black.svg" width="18px"> **Transcritos Brutos**: Conteúdo extraído originalmente dos PDFs/Docs.
- <img src="../../../../../assets/icons/human_16_black.svg" width="18px"> **Resumos Estratégicos**: Análises de P&D com foco em Defesa do Projeto e Norteamento de CAPEX.

### 2. [Bibliografia Consolidada (WTF_BIBLIOGRAPHY_CONSOLIDATED.md)](WTF_BIBLIOGRAPHY_CONSOLIDATED.md)
Lista completa e higienizada das obras de referência do projeto, catalogadas por título, autor e relevância temática.

## Protocolo de Atualização
- <img src="../../../../../assets/icons/human_11_black.svg" width="18px"> **Frequência**: O índice mestre deve ser atualizado sempre que novos transcritos ou resenhas forem gerados.
- <img src="../../../../../assets/icons/human_18_black.svg" width="18px"> **Ferramenta**: Utilize o script `.agents/scripts/generate_index.py` para sincronizar os links automaticamente.
- <img src="../../../../../assets/icons/human_08_black.svg" width="18px"> **Critério**: Apenas materiais técnicos e científicos validados devem constar neste catálogo. Documentos administrativos ou de triagem bruta (quarentena) não são indexados aqui.
